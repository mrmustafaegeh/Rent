## üåü Key Features

### üë§ For Customers
- **Global Reach**: Fully localized in English, Arabic (RTL support), Russian, Turkish, and Greek.
- **Easy Booking**: 3-step booking process with instant availability checks.
- **Smart Search**: Filter vehicles by brand, category (Luxury, SUV, Economy), and price.
- **User Dashboard**: Manage bookings, view history, and update profile settings.

## üöÄ Quick Start Guide

Prerequisites: Node.js 18+ and npm/yarn installed.

### 1. Installation
Clone the repository and install dependencies:
```bash
git clone https://github.com/your-username/rentalx.git
cd rentalx
npm install
```

### 2. Configuration
Create a `.env.local` file in the root directory based on `.env.example`.
*(See [TECHNICAL_DOCUMENTATION.md](./TECHNICAL_DOCUMENTATION.md) for full configuration details)*.

### 3. Run Development Server
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) to view the application.

### 4. Build for Production
```bash
npm run build
npm start
```
